üîç Model Architecture

Our custom 2-layer Convolutional Neural Network is optimized for malaria detection:

  1. Input Layer: 50√ó50 RGB normalized images
  2. Conv Block 1: 32 filters with ReLU activation + MaxPooling
  3. Conv Block 2: 64 filters with ReLU activation + MaxPooling
  4. Flatten Layer: Feature vector extraction
  5. Dense Layers: 128 units with dropout regularization
  6. Output Layer: Softmax for binary classification (Infected/Uninfected)

Optimization Details:

  • Optimizer: Adam (adaptive learning rate)
  • Loss Function: Categorical Cross-Entropy
  • Batch Size: 32
  • Epochs: Trained until convergence with early stopping
  • Dropout Rate: 0.5 (regularization)
